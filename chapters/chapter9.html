Chapter 9

Chi-Square Tests

Use chi-square tests to analyse categorical data — asking whether observed frequencies match expected ones, or whether two categorical variables are associated.

9.1 Goodness-of-Fit Test

Tests whether observed frequencies in one categorical variable match a set of expected (hypothesised) frequencies.

H₀: The observed frequencies match the expected distribution
χ² = Σ [(Oᵢ − Eᵢ)² / Eᵢ]   df = k − 1

where k = number of categories, O = observed count, E = expected count.

Assumption: Expected count ≥ 5 in each cell.

9.2 Test of Independence

Tests whether two categorical variables are independent in a contingency table.

H₀: Variable A and Variable B are independent (no association)
Eᵢⱼ = (row total × column total) / grand total
χ² = Σ [(Oᵢⱼ − Eᵢⱼ)² / Eᵢⱼ]   df = (rows−1)(cols−1)

9.4 Measures of Association: Odds Ratio and Relative Risk

A chi-square test tells us whether an association exists, but not its strength.

MeasureFormula (2×2 table)Used in
Odds Ratio (OR) (a×d) / (b×c) Case-control studies
Relative Risk (RR) [a/(a+b)] / [c/(c+d)] Cohort studies, RCTs
Risk Difference (ARD) a/(a+b) − c/(c+d) Absolute risk difference

Table: a=exposed cases, b=unexposed cases, c=exposed controls, d=unexposed controls (or similar 2×2 layout).
