Chapter 7

Hypothesis Testing

Hypothesis testing is the engine of statistical inference — a formal framework for deciding whether data provide enough evidence against a null hypothesis.

7.1 The Logic of Hypothesis Testing

Statistical hypothesis testing works by assuming the null hypothesis is true and asking: "How likely are we to see data at least as extreme as what we observed, purely by chance?"

  1. State the null hypothesis (H₀) and alternative hypothesis (H₁).
  2. Choose a significance level α (typically 0.05).
  3. Collect data and compute a test statistic.
  4. Calculate the p-value.
  5. Make a decision: reject H₀ if p < α.
  6. Interpret the result in context.

7.2 Hypotheses

The null hypothesis (H₀) is the default claim of "no effect" or "no difference." The alternative hypothesis (H₁) is what we hope to show.

Test typeH₀H₁
Two-tailedμ = μ₀μ ≠ μ₀
Right-tailedμ ≤ μ₀μ > μ₀
Left-tailedμ ≥ μ₀μ < μ₀

7.3 The p-value

The p-value is the probability of obtaining a test statistic at least as extreme as the one computed, assuming H₀ is true.

⚠️ Common Misconception The p-value is not the probability that H₀ is true. It is the probability of the observed data (or more extreme) if H₀ were true.
  • p < α → statistically significant evidence against H₀ → reject H₀
  • p ≥ α → insufficient evidence against H₀ → fail to reject H₀
💡 Statistical vs Clinical Significance A result can be statistically significant (small p-value) but clinically unimportant (very small effect size). Always report and interpret effect sizes alongside p-values.

7.4 Type I and Type II Errors

H₀ is TRUEH₀ is FALSE
Reject H₀❌ Type I error (False Positive) P = α✅ Correct (Power) P = 1 − β
Fail to reject H₀✅ Correct P = 1 − α❌ Type II error (False Negative) P = β
  • α (Type I error rate): set by the researcher; typically 0.05.
  • β (Type II error rate): related to sample size and effect size; typically 0.20.
  • Power = 1 − β: probability of correctly detecting a real effect.
